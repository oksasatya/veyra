# Veyra MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the complete Veyra REST API — a self-hosted vehicle management backend — from zero to a working Docker Compose deployment with auth, vehicle CRUD, service history, fuel logs, expenses, reminders, documents, and a dashboard summary endpoint.

**Architecture:** Enhanced single-crate Hexagonal DDD. Domain layer owns all business rules and imports nothing from axum/sqlx/serde. Application layer orchestrates use cases via async port traits. Adapters implement those ports (Postgres outbound, axum inbound). CI script enforces layer boundaries.

**Tech Stack:** Rust 2021 · axum 0.8 · sqlx 0.8 (compile-time checked, postgres) · PostgreSQL 17 · argon2 0.5 · jsonwebtoken 9 · thiserror 2 · figment 0.10 · Docker Compose · cargo nextest · testcontainers-modules 0.4

## Global Constraints

- Rust edition 2021; MSRV 1.82 (`rust-toolchain.toml` pins this)
- `#![forbid(unsafe_code)]` at crate root — no exceptions
- No `.unwrap()` / `.expect()` / `panic!` / `todo!()` on production code paths — use `Result` + `?`
- `thiserror` for all domain + application error enums; `anyhow::Result<()>` at `main.rs` boundary only
- `clippy::too_many_arguments` — ≤7 params (target ≤5); extract params struct at 5+
- `clippy::cognitive_complexity` — ≤15; extract named helpers
- All sqlx queries parameterized (`$1`, `$2`, …) — never `format!()` into SQL
- Every handler scopes queries by `user_id` extracted from JWT — no cross-user data leakage
- Error response shape always: `{"error":{"code":"SCREAMING_SNAKE","message":"lowercase detail"}}`
- Verify before "done": `cargo fmt --check → cargo clippy --all-targets --all-features -- -D warnings → cargo nextest run`

- [ ] **Step 10: Create `apps/backend/src/main.rs`**

```rust
use anyhow::Result;
use sqlx::postgres::PgPoolOptions;
use std::net::SocketAddr;
use tracing_subscriber::{layer::SubscriberExt, util::SubscriberInitExt};
use veyra::adapters::inbound::http::router;
use veyra::bootstrap::{config::Config, state::AppState};

#[tokio::main]
async fn main() -> Result<()> {
    tracing_subscriber::registry()
        .with(tracing_subscriber::EnvFilter::try_from_default_env()
            .unwrap_or_else(|_| "veyra=debug".into()))
        .with(tracing_subscriber::fmt::layer())
        .init();

    let config = Config::load()?;

    let pool = PgPoolOptions::new()
        .max_connections(10)
        .connect(&config.database_url)
        .await?;

    sqlx::migrate!("./migrations").run(&pool).await?;

    let state = AppState::new(pool, config.jwt_secret);
    let app = router::build(state);

    let addr = SocketAddr::from(([0, 0, 0, 0], config.port));
    tracing::info!("listening on {}", addr);

    let listener = tokio::net::TcpListener::bind(addr).await?;
    axum::serve(listener, app).await?;

    Ok(())
}
```

#[derive(Debug, Clone, PartialEq)]
pub struct Email(String);

impl Email {
    pub fn new(raw: String) -> Result<Self, DomainError> {
        let normalized = raw.trim().to_lowercase();
        if normalized.is_empty() || !normalized.contains('@') {
            return Err(DomainError::InvalidEmail(raw));
        }
        let parts: Vec<&str> = normalized.splitn(2, '@').collect();
        if parts[0].is_empty() || parts[1].is_empty() || !parts[1].contains('.') {
            return Err(DomainError::InvalidEmail(raw));
        }
        Ok(Self(normalized))
    }

    pub fn as_str(&self) -> &str {
        &self.0
    }

    pub async fn execute(
        &self,
        email: String,
        password: String,
        name: String,
    ) -> Result<String, AppError> {
        // Validate email
        let email_vo = Email::new(email.clone()).map_err(AppError::from)?;

        // Validate password length
        if password.len() < 8 {
            return Err(AppError::Validation(
                DomainError::PasswordTooShort.to_string(),
            ));
        }

        // Check uniqueness
        match self.user_repo.find_by_email(email_vo.as_str()).await {
            Ok(_) => return Err(AppError::Conflict("email already registered".into())),
            Err(crate::ports::repositories::RepositoryError::NotFound) => {}
            Err(e) => return Err(AppError::from(e)),
        }

        // Hash password
        let salt = SaltString::generate(&mut OsRng);
        let hash = Argon2::default()
            .hash_password(password.as_bytes(), &salt)
            .map_err(|e| AppError::Internal(e.to_string()))?
            .to_string();

        // Insert user
        let user = self
            .user_repo
            .insert(email_vo.as_str(), &hash, &name)
            .await
            .map_err(AppError::from)?;

        // Sign JWT
        self.auth
            .sign_token(user.id)
            .map_err(|e| AppError::Internal(e.to_string()))
    }

use chrono::{Duration, Utc};
use jsonwebtoken::{decode, encode, Algorithm, DecodingKey, EncodingKey, Header, Validation};
use serde::{Deserialize, Serialize};
use uuid::Uuid;

use crate::ports::auth::{AuthError, AuthPort};

#[derive(Debug, Serialize, Deserialize)]
struct Claims {
    sub: String,
    exp: i64,
}

#[derive(Clone)]
pub struct JwtAuth {
    secret: String,
}

impl JwtAuth {
    pub fn new(secret: String) -> Self {
        Self { secret }
    }
}

impl AuthPort for JwtAuth {
    fn sign_token(&self, user_id: Uuid) -> Result<String, AuthError> {
        let exp = (Utc::now() + Duration::days(7)).timestamp();
        let claims = Claims { sub: user_id.to_string(), exp };
        encode(
            &Header::default(),
            &claims,
            &EncodingKey::from_secret(self.secret.as_bytes()),
        )
        .map_err(|e| AuthError::SigningFailed(e.to_string()))
    }

    fn verify_token(&self, token: &str) -> Result<Uuid, AuthError> {
        let token_data = decode::<Claims>(
            token,
            &DecodingKey::from_secret(self.secret.as_bytes()),
            &Validation::new(Algorithm::HS256),
        )
        .map_err(|_| AuthError::InvalidToken)?;

        token_data
            .claims
            .sub
            .parse::<Uuid>()
            .map_err(|_| AuthError::InvalidToken)
    }
}

        let row = sqlx::query_as!(
            UserRow,
            "SELECT id, email, password_hash, name, created_at, updated_at FROM users WHERE email = $1",
            email
        )
        .fetch_optional(&self.pool)
        .await?
        .ok_or(RepositoryError::NotFound)?;

        row_to_user(row)
    }

    async fn find_by_id(&self, id: Uuid) -> RepositoryResult<User> {
        let row = sqlx::query_as!(
            UserRow,
            "SELECT id, email, password_hash, name, created_at, updated_at FROM users WHERE id = $1",
            id
        )
        .fetch_optional(&self.pool)
        .await?
        .ok_or(RepositoryError::NotFound)?;

        row_to_user(row)
    }

    async fn insert(&self, email: &str, password_hash: &str, name: &str) -> RepositoryResult<User> {
        let row = sqlx::query_as!(
            UserRow,
            r#"INSERT INTO users (email, password_hash, name)
               VALUES ($1, $2, $3)
               RETURNING id, email, password_hash, name, created_at, updated_at"#,
            email,
            password_hash,
            name
        )
        .fetch_one(&self.pool)
        .await
        .map_err(|e| {
            if let sqlx::Error::Database(ref db_err) = e {
                if db_err.constraint() == Some("users_email_key") {
                    return RepositoryError::Conflict("email already registered".into());
                }
            }
            RepositoryError::Database(e)
        })?;

        row_to_user(row)
    }

pub async fn require_auth(
    State(state): State<AppState>,
    mut req: Request,
    next: Next,
) -> Result<Response, StatusCode> {
    let token = req
        .headers()
        .get("Authorization")
        .and_then(|v| v.to_str().ok())
        .and_then(|v| v.strip_prefix("Bearer "))
        .ok_or(StatusCode::UNAUTHORIZED)?;

    let user_id = state
        .auth
        .verify_token(token)
        .map_err(|_| StatusCode::UNAUTHORIZED)?;

    req.extensions_mut().insert(user_id);
    Ok(next.run(req).await)
}

fn row_to_vehicle(row: VehicleRow) -> Result<Vehicle, RepositoryError> {
    Ok(Vehicle {
        id: row.id,
        user_id: row.user_id,
        brand: row.brand,
        model: row.model,
        year: row.year,
        plate_number: PlateNumber::new(row.plate_number)
            .map_err(|e| RepositoryError::Database(sqlx::Error::Protocol(e.to_string())))?,
        color: row.color,
        fuel_type: FuelType::from_str(&row.fuel_type)
            .map_err(|e| RepositoryError::Database(sqlx::Error::Protocol(e.to_string())))?,
        current_odometer: Odometer::new(row.current_odometer as u32),
        notes: row.notes,
        created_at: row.created_at,
        updated_at: row.updated_at,
    })
}

#[async_trait]
impl VehicleRepository for PgVehicleRepo {
    async fn list_by_user(&self, user_id: Uuid) -> RepositoryResult<Vec<Vehicle>> {
        let rows = sqlx::query_as!(
            VehicleRow,
            r#"SELECT id, user_id, brand, model, year, plate_number, color, fuel_type,
                      current_odometer, notes, created_at, updated_at
               FROM vehicles WHERE user_id = $1 ORDER BY created_at DESC"#,
            user_id
        )
        .fetch_all(&self.pool)
        .await?;

        rows.into_iter().map(row_to_vehicle).collect()
    }

    async fn find_by_id(&self, id: Uuid, user_id: Uuid) -> RepositoryResult<Vehicle> {
        let row = sqlx::query_as!(
            VehicleRow,
            r#"SELECT id, user_id, brand, model, year, plate_number, color, fuel_type,
                      current_odometer, notes, created_at, updated_at
               FROM vehicles WHERE id = $1 AND user_id = $2"#,
            id,
            user_id
        )
        .fetch_optional(&self.pool)
        .await?
        .ok_or(RepositoryError::NotFound)?;

        row_to_vehicle(row)
    }

    async fn insert(&self, params: CreateVehicleParams) -> RepositoryResult<Vehicle> {
        let row = sqlx::query_as!(
            VehicleRow,
            r#"INSERT INTO vehicles (user_id, brand, model, year, plate_number, color, fuel_type, current_odometer, notes)
               VALUES ($1, $2, $3, $4, $5, $6, $7, $8, $9)
               RETURNING id, user_id, brand, model, year, plate_number, color, fuel_type,
                         current_odometer, notes, created_at, updated_at"#,
            params.user_id, params.brand, params.model, params.year,
            params.plate_number, params.color, params.fuel_type,
            params.current_odometer as i32, params.notes
        )
        .fetch_one(&self.pool)
        .await
        .map_err(|e| {
            if let sqlx::Error::Database(ref db_err) = e {
                if db_err.constraint() == Some("vehicles_user_id_plate_number_key") {
                    return RepositoryError::Conflict("plate number already registered for this account".into());
                }
            }
            RepositoryError::Database(e)
        })?;

        row_to_vehicle(row)
    }

    async fn update(&self, id: Uuid, user_id: Uuid, params: UpdateVehicleParams) -> RepositoryResult<Vehicle> {
        let row = sqlx::query_as!(
            VehicleRow,
            r#"UPDATE vehicles SET brand=$3, model=$4, year=$5, color=$6, fuel_type=$7,
                                   current_odometer=$8, notes=$9, updated_at=NOW()
               WHERE id=$1 AND user_id=$2
               RETURNING id, user_id, brand, model, year, plate_number, color, fuel_type,
                         current_odometer, notes, created_at, updated_at"#,
            id, user_id, params.brand, params.model, params.year,
            params.color, params.fuel_type, params.current_odometer as i32, params.notes
        )
        .fetch_optional(&self.pool)
        .await?
        .ok_or(RepositoryError::NotFound)?;

        row_to_vehicle(row)
    }

    async fn delete(&self, id: Uuid, user_id: Uuid) -> RepositoryResult<()> {
        let result = sqlx::query!(
            "DELETE FROM vehicles WHERE id = $1 AND user_id = $2",
            id, user_id
        )
        .execute(&self.pool)
        .await?;

        if result.rows_affected() == 0 {
            return Err(RepositoryError::NotFound);
        }
        Ok(())
    }
}

use sqlx::PgPool;
use uuid::Uuid;
use rust_decimal::Decimal;
use crate::application::errors::AppError;

#[derive(Debug)]
pub struct VehicleSummary {
    pub vehicle_id: Uuid,
    pub current_odometer: u32,
    pub total_services: i64,
    pub total_service_cost: Decimal,
    pub total_refuels: i64,
    pub total_fuel_cost: Decimal,
    pub total_expenses: Decimal,
    pub upcoming_reminders: i64,
}

pub struct GetSummaryUseCase {
    pub pool: PgPool,
}

impl GetSummaryUseCase {
    pub async fn execute(&self, vehicle_id: Uuid, user_id: Uuid) -> Result<VehicleSummary, AppError> {
        let row = sqlx::query!(
            r#"SELECT
                v.id,
                v.current_odometer,
                COUNT(DISTINCT s.id)                                              AS "total_services!: i64",
                COALESCE(SUM(s.cost), 0)                                          AS "total_service_cost!: Decimal",
                COUNT(DISTINCT f.id)                                              AS "total_refuels!: i64",
                COALESCE(SUM(f.total_cost), 0)                                    AS "total_fuel_cost!: Decimal",
                COALESCE(SUM(e.amount), 0)                                        AS "total_expenses!: Decimal",
                COUNT(DISTINCT r.id) FILTER (
                    WHERE r.is_completed = FALSE
                      AND (
                        r.due_date     <= CURRENT_DATE + INTERVAL '30 days'
                        OR r.due_odometer <= v.current_odometer + 500
                      )
                )                                                                 AS "upcoming_reminders!: i64"
            FROM vehicles v
            LEFT JOIN service_records s ON s.vehicle_id = v.id
            LEFT JOIN fuel_logs       f ON f.vehicle_id = v.id
            LEFT JOIN expenses        e ON e.vehicle_id = v.id
            LEFT JOIN reminders       r ON r.vehicle_id = v.id
            WHERE v.id = $1 AND v.user_id = $2
            GROUP BY v.id, v.current_odometer"#,
            vehicle_id,
            user_id
        )
        .fetch_optional(&self.pool)
        .await
        .map_err(|e| AppError::Internal(e.to_string()))?
        .ok_or(AppError::NotFound)?;

        Ok(VehicleSummary {
            vehicle_id: row.id,
            current_odometer: row.current_odometer as u32,
            total_services: row.total_services,
            total_service_cost: row.total_service_cost,
            total_refuels: row.total_refuels,
            total_fuel_cost: row.total_fuel_cost,
            total_expenses: row.total_expenses,
            upcoming_reminders: row.upcoming_reminders,
        })
    }
}

```bash
#!/usr/bin/env bash
set -euo pipefail

DOMAIN="apps/backend/src/domain"
APP="apps/backend/src/application"
PORTS="apps/backend/src/ports"

FORBIDDEN_IN_DOMAIN=("axum" "sqlx" "tokio" "serde")
FORBIDDEN_IN_APP=("axum" "sqlx")
FORBIDDEN_IN_PORTS=("axum" "sqlx")

check() {
    local dir="$1"
    shift
    local forbidden=("$@")
    local found=0

    for pkg in "${forbidden[@]}"; do
        matches=$(grep -rn "use ${pkg}" "$dir" 2>/dev/null || true)
        if [[ -n "$matches" ]]; then
            echo "❌ BOUNDARY VIOLATION: '$pkg' imported in $dir"
            echo "$matches"
            found=1
        fi
    done
    return $found
}

echo "Checking hexagonal layer boundaries..."
check "$DOMAIN" "${FORBIDDEN_IN_DOMAIN[@]}"
check "$APP"    "${FORBIDDEN_IN_APP[@]}"
check "$PORTS"  "${FORBIDDEN_IN_PORTS[@]}"
echo "✅ All boundary checks passed"
```
